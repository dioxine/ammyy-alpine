# Служба Ammyy Relay & Router для Alpine Linux

Сменить язык: 🇬🇧 [English Version](README.md)

---

Легковесный слой совместимости glibc для запуска бинарных файлов Ammyy напрямую в Alpine Linux с использованием высокооптимизированного, готового к продакшену минимального Docker-контейнера.

> [!NOTE]
> Чистый размер контента: **8.86 МБ** | Занимаемое место на диске: **29.9 МБ**

## Сетевая карта портов по умолчанию

По умолчанию стек использует альтернативные порты, что полностью исключает конфликты с существующими веб-серверами (такими как Apache или Nginx):

| Порт хоста | Протокол |    Компонент   | Назначение                                                      |
|:-----------|:---------|:---------------|:----------------------------------------------------------------|
| **8080**   | TCP      | `ammyy_router` | Основной рабочий порт роутера для клиентских подключений        |
| **8443**   | TCP      | `ammyy_router` | Альтернативный рабочий порт роутера (вместо стандартного `443`) |
| **85**     | TCP      | `ammyy_router` | Встроенная веб-консоль управления роутера                       |
| **3080**   | TCP      | `ammyy_relay`  | Рабочий порт реле (вместо стандартного веб-порта `80`)          |

## Быстрое развертывание через Docker Compose

> [!IMPORTANT]
> Для управления демонами Dockerруют требуются права суперпользователя. Убедитесь, что вы запущены под учетной записью `root` или используете `sudo`.

### 1. Скачивание файлов конфигурации
Создайте рабочую директорию на хосте и загрузите структурированные файлы настроек напрямую на ваш сервер:

```bash
mkdir -p /opt/ammyy/config /opt/ammyy/logs
curl -fsSL https://githubusercontent.com -o /opt/ammyy/config/router.cfg
curl -fsSL https://githubusercontent.com -o /opt/ammyy/config/relay.cfg
curl -fsSL https://githubusercontent.com -o /opt/ammyy/docker-compose.yml
```

### 2. Запуск стека служб
Перейдите в рабочую папку и поднимите микро-контейнер в фоновом режиме:

```bash
cd /opt/ammyy
docker compose up -d
```
*Docker самостоятельно скачает оптимизированный образ `gdioxine/ammyy-stack:latest` с Docker Hub, настроит изолированную сеть и пробросит порты.*

## Структура рабочей директории

Приложение изолирует исполняемую среду внутри слепка контейнера, оставляя на хост-машине только текстовые конфигурации и логи:

```text
/opt/ammyy/
├── docker-compose.yml     # Файл оркестрации стека служб
├── config/                # Изменяемые файлы конфигураций
│   ├── relay.cfg          # Сетевые настройки и порты реле
│   └── router.cfg         # Активный конфигурационный...
```
